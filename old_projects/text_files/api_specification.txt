HTTP метод GET
Путь /products
Описание: получение списка всех товаров (пагинация: список приходит частями)
Path параметры: отсутствуют
Query параметры: limit - количество товаров, offset - смещение по товарам
Статусы ответов: успех [200 - ок] ошибка [500 - ошибка сервера]
Пример запроса и ответа:
GET /products?limit=10&offset=0
ответ 200 = ок + [{"id": 1,"name": "Ноутбук","price": 50000,"quantity": 10},] (обработанный фронтом)


HTTP метод GET
Путь /products/{id}
Описание: получение товара по ID
Path параметры: id - идентификатор товара
Query параметры: отсутствуют
Статусы ответов: успех [200 - ок] ошибка [404 - товар не найден, 500 - ошибка сервера]
Пример запроса и ответа:
GET /products/1
ответ 200 = {"id":1,"name":"Ноутбук","price":50000,"quantity":10}
ответ 404 = {"error":"Товар не найден"}


HTTP метод POST
Путь /products
Описание: создание нового товара
Path параметры: отсутствуют
Query параметры: отсутствуют
Тело запроса: name - название товара (обязательное), price - цена (обязательное), quantity - количество (необязательное)
Статусы ответов: успех [201 - товар создан] ошибка [400 - неправильные данные, 500 - ошибка сервера]
Пример запроса и ответа:
POST /products
тело = {"name":"Клавиатура","price":3000,"quantity":15}
ответ 201 = {"id":3,"name":"Клавиатура","price":3000,"quantity":15}


HTTP метод PUT
Путь /products/{id}
Описание: обновление товара
Path параметры: id - идентификатор товара
Query параметры: отсутствуют
Тело запроса: name - новое название (необязательное), price - новая цена (необязательное), quantity - новое количество (необязательное)
Статусы ответов: успех [200 - товар обновлен] ошибка [400 - неправильные данные, 404 - товар не найден, 500 - ошибка сервера]
Пример запроса и ответа:
PUT /products/1
тело = {"price":45000}
ответ 200 = {"id":1,"name":"Ноутбук","price":45000,"quantity":10}

HTTP метод POST
Путь /orders
Описание: создание нового заказа
Path параметры: отсутствуют
Query параметры: отсутствуют
Тело запроса: user_id - ID пользователя, product_id - ID товара, quantity - количество
Статусы ответов: успех [201 - заказ создан] ошибка [400 - неправильные данные, 500 - ошибка сервера]
